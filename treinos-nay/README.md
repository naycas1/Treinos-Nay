# Treinos da Nay 💪

App pessoal de treinos com checklist, nutrição e notificações.

## Como subir no GitHub e Vercel pelo celular

### PASSO 1 — Criar repositório no GitHub
1. Abre github.com no celular
2. Clica no **+** no canto superior direito
3. Clica em **New repository**
4. Nome: `treinos-nay`
5. Deixa como **Public**
6. Clica em **Create repository**

### PASSO 2 — Fazer upload dos arquivos
1. No repositório criado, clica em **uploading an existing file**
2. Sobe os arquivos nesta ordem:
   - `package.json`
   - `public/index.html`
   - `public/manifest.json`
   - `public/sw.js`
   - `src/index.js`
   - `src/App.js`
3. Clica em **Commit changes**

### PASSO 3 — Subir no Vercel
1. Abre vercel.com no celular
2. Cria conta com o GitHub (botão "Continue with GitHub")
3. Clica em **Add New Project**
4. Seleciona o repositório `treinos-nay`
5. Clica em **Deploy**
6. Aguarda ~2 minutos

### PASSO 4 — Instalar na tela inicial
1. Abre o link gerado pelo Vercel no **Safari** (iPhone) ou **Chrome** (Android)
2. iPhone: clica no ícone de compartilhar → "Adicionar à Tela de Início"
3. Android: clica nos 3 pontos → "Adicionar à tela inicial"

Pronto! O app fica na tela inicial igual um app nativo. 🎉
